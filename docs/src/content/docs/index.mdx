<CardGrid stagger>

    <Card title="YouTube" icon="youtube">

    	[Tweet&nbsp;docs&nbsp;→](/components/twitter/)

    	```mdx title="example.mdx"
    	https://www.youtube.com/watch?v=TtRtkTzHVBU

    	```

    	[Learn how to configure auto-embeds&nbsp;→](/getting-started/#auto-embed-urls-in-mdx)

    </Card>
